# Lunar Ephemeris

An open-source Progressive Web Application (PWA) that provides accurate
lunar phase information, lunar cycle data, and local astronomical events
using established astronomical calculation libraries. The application is
designed to run entirely in the browser, supports offline operation, and
can be installed on desktop and mobile devices.

**Version:** 1.0.0

------------------------------------------------------------------------

## Overview

Lunar Ephemeris is a client-side React application that combines
established astronomical libraries with a modern web interface to
provide daily lunar information without requiring a backend server or
user account.

The application was intentionally designed to perform all calculations
locally within the browser. This approach provides transparency, fast
performance, privacy, and offline capability while avoiding dependence
on external APIs for routine astronomical calculations.

------------------------------------------------------------------------

## Features

### Daily Lunar Summary

Displays current lunar information including:

-   Moon phase
-   Illumination percentage
-   Lunar age
-   Current zodiac position (where applicable)
-   Additional daily lunar information

------------------------------------------------------------------------

### Dashboard

Interactive dashboard presenting the most important astronomical
information for the selected date.

------------------------------------------------------------------------

### Monthly Lunar Calendar

Monthly calendar displaying lunar phases throughout the month for quick
planning and reference.

------------------------------------------------------------------------

### Local Astronomical Events

Calculates local events using the user's location, including appropriate
solar and lunar event information.

------------------------------------------------------------------------

### Time Zone Support

Calculations may be viewed using selectable time zones, allowing users
to examine lunar information for different regions.

------------------------------------------------------------------------

### Geolocation

Optionally retrieves the user's current location to improve local
astronomical calculations.

No location information is transmitted to a server.

------------------------------------------------------------------------

### Shareable URLs

Current application state can be shared using URL parameters.

Recipients opening the link see the same selected date, view, and
configuration.

------------------------------------------------------------------------

### Progressive Web App (PWA)

The application supports installation on compatible desktop and mobile
browsers.

Features include:

-   Installable application
-   Offline functionality
-   Service Worker
-   Web App Manifest
-   Application icons
-   Cached assets for offline use

------------------------------------------------------------------------

### Offline Operation

Once loaded, the application continues functioning without an Internet
connection using cached assets generated by the Service Worker.

## Installation

Clone the repository:

``` bash
git clone https://github.com/IainAmosMelchizedek/lunar-ephemeris.git
```

Install dependencies:

``` bash
npm install
```

Run the development server:

``` bash
npm run dev
```

Create a production build:

``` bash
npm run build
```

Preview the production build:

``` bash
npm run preview
```
